-------------------------------------------------------------------------------------------------------
    Data flow
-------------------------------------------------------------------------------------------------------
+ Angularjs (1.x) uses Two-way data binding/flow and it cause performance problem.
+ New Angular (2.x +) uses One-way data binding/flow similar to ReactJS or VueJS. 
+ However, we can still implement two-way binding manually by adding some additional logic.
