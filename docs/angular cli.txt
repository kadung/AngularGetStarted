---------------------------------------------------------------------------------------------------
    Overview
---------------------------------------------------------------------------------------------------
- The Angular CLI is a tool to initialize, develop, scaffold and maintain Angular applications
- To install the Angular CLI:
    npm install -g @angular/cli

---------------------------------------------------------------------------------------------------
    Commands
---------------------------------------------------------------------------------------------------
+ Create new Angular project:  ng new [project name]
    - Ref: https://github.com/angular/angular-cli/wiki/new

+ Builds the application and starts a web server:  ng serve
    - Navigate to http://localhost:4200/ and the app will automatically reload if any change.
    - Ref: https://github.com/angular/angular-cli/wiki/serve
    
+ Running unit tests:  ng test
    - Tests will execute after a build is executed via Karma, and it will automatically watch your 
    files for changes. 
    - You can run tests a single time via --watch=false or --single-run.

+ Running end-to-end tests:  ng e2e
    - Before running the tests make sure you are serving the app via ng serve. 
    - End-to-end tests are run via Protractor.
